## 公告聊天网站
（源码）https://github.com/oxogenesis/oxo-chat-server  
（网站）https://oxo-chat-server.com/  
对上面网站代码进行优化完善，按照论坛或者推特（无用户注册管理模块，只能浏览公告，不能发布公告，发布需要用到上面应用）模式，丰富搜索、排行（请求量、引用量、账号权重等）、推荐等功能，界面简洁友好，请先star、fork。
